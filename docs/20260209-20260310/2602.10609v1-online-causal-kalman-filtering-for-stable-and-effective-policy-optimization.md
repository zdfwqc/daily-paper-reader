## 摘要
大语言模型的强化学习面临高方差的词元级（token-level）重要性采样（IS）比率问题，这在大规模应用时会使策略优化变得不稳定。为了提高稳定性，近期的方法通常对序列中的所有词元使用固定的序列级 IS 比率，或者分别调整每个词元的 IS 比率，从而忽略了序列中词元间的时序离策（off-policy）偏差。在本文中，我们首先通过实验发现，局部离策偏差在词元层面存在结构性不一致，这可能会扭曲相邻词元间的策略梯度更新，并导致训练崩溃。为解决这一问题，我们提出了用于稳定且有效策略优化的在线因果卡尔曼滤波（KPO）。具体而言，我们将目标 IS 比率建模为随词元演化的隐状态，并应用卡尔曼滤波器，根据过去词元的状态在线且自回归地更新该状态，而不依赖于未来词元。过滤后的 IS 比率保留了词元级的局部结构感知变化，同时强力平滑了噪声峰值，从而实现了更稳定、更有效的策略更新。实验表明，与现有最先进的方法相比，KPO 在具有挑战性的数学推理数据集上取得了更优异的结果。

## Abstract
Reinforcement learning for large language models suffers from high-variance token-level importance sampling (IS) ratios, which would destabilize policy optimization at scale. To improve stability, recent methods typically use a fixed sequence-level IS ratio for all tokens in a sequence or adjust each token's IS ratio separately, thereby neglecting temporal off-policy derivation across tokens in a sequence. In this paper, we first empirically identify that local off-policy deviation is structurally inconsistent at the token level, which may distort policy-gradient updates across adjacent tokens and lead to training collapse. To address the issue, we propose Online Causal Kalman Filtering for stable and effective Policy Optimization (KPO). Concretely, we model the desired IS ratio as a latent state that evolves across tokens and apply a Kalman filter to update this state online and autoregressively based on the states of past tokens, regardless of future tokens. The resulting filtered IS ratios preserve token-wise local structure-aware variation while strongly smoothing noise spikes, yielding more stable and effective policy updates. Experimentally, KPO achieves superior results on challenging math reasoning datasets compared with state-of-the-art counterparts.

论文针对大语言模型（LLM）在强化学习（RL）后训练阶段（如使用 GRPO 算法时）的**训练不稳定性**问题展开研究。
*   **研究背景**：在大规模 RL 训练中，词元级（token-level）的重要性采样（IS）比率往往存在极高的方差，容易导致策略梯度扭曲、熵崩溃（Entropy Collapse）以及训练中断。
*   **核心发现**：作者通过实证分析发现，token 级的离策（off-policy）偏差在局部上是“结构不一致”的——即相邻 token 的 IS 比率波动剧烈且频繁切换，这违背了“相邻语义片段应具有相似离策程度”的直觉。
*   **研究动机**：现有的序列级平滑方法（如 GSPO）虽然稳定但丢失了局部结构信息，而 token 级方法又过于嘈杂。因此，需要一种既能平滑噪声又能保留局部时序结构的方法。

### 2. 方法论：Online Causal Kalman Filtering (KPO)
KPO 的核心思想是将 token 级 IS 比率建模为一个**状态空间模型**，并利用卡尔曼滤波进行在线平滑。

*   **核心建模**：将目标 IS 比率视为一个随 token 位置演化的隐状态（Latent State）。为了数值稳定性，滤波过程在**对数空间**（log space）中进行。
*   **关键技术细节**：
    1.  **预测步骤（Prediction）**：基于前一个 token 的状态预测当前状态，引入过程噪声 $Q$ 来允许状态随序列逐渐演化。
    2.  **卡尔曼增益（Kalman Gain）**：计算预测值与观测值（原始 IS 比率）之间的权重。如果观测噪声 $V$ 较大，则更信任预测值；反之则更信任当前观测。
    3.  **更新步骤（Update）**：利用当前 token 的观测值修正预测，得到平滑后的隐状态，并更新误差协方差。
    4.  **因果性（Causality）**：该滤波器是自回归的，仅依赖当前及过去的 token 信息，符合 LLM 的生成逻辑。
*   **目标函数**：将过滤后的 IS 比率代入 PPO/GRPO 的剪切代理目标函数中，实现更稳定的梯度更新。

### 6. 主要结论与发现
*   **性能提升**：KPO 在几乎所有数学基准上均优于 GRPO、GMPO 和 GSPO。特别是在 AIME 等长链条推理任务中，提升尤为显著。
*   **稳定性增强**：KPO 有效缓解了 GRPO 常见的熵崩溃问题，使模型在训练后期仍能保持探索能力。
*   **结构保留**：卡尔曼滤波成功将 token 动态从“高频噪声模式”转变为“长程相干段”，这证明了局部离策一致性对策略优化的重要性。
*   **剪切机制协同**：带剪切的 KPO（KPO-clipped）效果最好，说明滤波平滑与 PPO 剪切在处理极端离策值时具有互补作用。

### 7. 优点
*   **视角新颖**：首次从时序信号处理的角度（卡尔曼滤波）审视 token 级重要性采样，解决了局部一致性缺失的问题。
*   **即插即用**：方法轻量，不改变模型架构，易于集成到现有的 RLHF 框架（如 DeepSpeed-Chat 或 LLaMA-Factory）中。
*   **理论支撑**：利用成熟的控制理论解决深度强化学习中的方差问题，比启发式的软门控或硬剪切更具解释性。

### 8. 不足与局限
*   **并行化挑战**：由于卡尔曼滤波是自回归（顺序）处理 token 的，在计算 IS 比率时难以像标准 Transformer 推理那样完全并行化，可能在大规模序列上带来微小的计算开销。
*   **任务局限性**：实验主要集中在数学推理任务上，尚不清楚该方法在创意写作、多轮对话或代码生成等其他 RLHF 场景下的表现。
*   **超参数依赖**：虽然论文给出了推荐值，但 $Q$ 和 $V$ 的选择可能仍需根据具体模型和任务进行微调，以平衡“平滑度”与“响应速度”。
